# sbpe-terminal
SBPE (Set Based Parameter Explorer) is a terminal-based prototype for exploring and analyzing 2D parameter spaces with threshold-based feasible region detection. 
SBPE (Set-Based Parameter Explorer) は、セットベース設計の考え方をもとに、2次元設計空間におけるパラメータ探索を行うターミナルベースのプロトタイプです。

単一の最適解ではなく、条件を満たす成立領域（feasible region）を探索し、パラメータ感度や妥協点を把握することを目的としています。

現在は、
関数値の数値マップ、
最適値、
許容範囲面積などを
ターミナル上へ可視化する段階です。
