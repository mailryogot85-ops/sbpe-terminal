<概要>

SBPE（Set-Based Parameter Explorer）は、セットベース設計の考え方をもとに、2次元パラメータ空間を探索・可視化するターミナルベースのプロトタイプです。
単一の最適解を探すだけでなく、許容領域を把握することを目的としています。


<開発背景>

OpenCVを用いた白線検知システムを開発していた時に、Cannyエッジ検出のHough変換の時にパラメータ調整が一回一回手動で煩雑でした。
そこで、大学講義で学んだ、セットベース設計の考え方から影響を受け、一つの最適点ではなく、成立可能な領域を探索・可視化するSBPEの構想を始めました。


<現在の実装>

まだまだこれからですが、現在は草案・プロトタイプとして、
・変数二次関数
・ガウシアンを用いたスコア化(0.00~1.00)
・閾値を満たす許容範囲のスコアの面積を算出
・terminal上での score map 出力
に対応させました。


<今後の展望>

今後は、
・読み込んだ画像からevaluation関数を生成したり、やそれに準する評価基準を導入
・ターゲットとなる最適解と許容範囲の算出
・OpenCVとの統合
・３次元以上の多変数パラメータ探索
・連続的なヒートマップのような形式で出力
などを目指しています。
